# Meat - Angular App

## Instructions

### 1. Cloning Repository 

`git clone https://github.com/cod3rcursos/meat-app-starter.git`

### 2. Installing Dependencies 

`npm install`

### 3. Starting Server

`ng serve` ou `npm start`

### 4. Installing json-server

`npm install -g json-server`

### 5. Start service (app root)

`json-server db.json`
