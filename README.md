# Meat - Angular App

## 0. Installing Dependencies 

`npm install

## 1. Starting Server

`ng serve` ou `npm start`

## 2. Starting Backend

### Installing json-server

`npm install -g json-server`

### Start srevice (app root)

`json-server db.json`
